# DESIGN

## 1. 项目目标
使用 Python 面向对象实现一个控制台学生管理系统。

## 2. 核心对象
### Student
- 表示单个学生
- 属性：学号、姓名、年龄、性别、成绩

### StudentManager
- 管理多个 Student 对象
- 负责添加、删除、修改、查找、显示学生信息

## 3. 程序入口
### main
- 显示菜单
- 接收用户输入
- 调用 StudentManager 的方法

## 4. 当前设计决策
- 使用列表保存学生对象
- 学号唯一
- 先做控制台版本
- 先不做文件持久化

## 5. 后续可扩展方向
- 保存到 JSON
- 排序功能
- 统计功能
