# Week 1 Review

## 本周学了什么

这一周主要学习了以下内容：

- Git / GitHub 基础
- Linux / WSL 常用命令
- Python 虚拟环境与依赖管理
- Python 项目结构
- Git 常见工作流
- Markdown 基础
- README 和学习笔记整理方法

---

## 本周完成了什么

这一周完成了以下实际内容：

- 配置了 Git 用户名、邮箱和 SSH
- 创建了学习仓库 `ai-agent-learning`
- 在 WSL 中完成了基础命令练习
- 创建并整理了多个学习笔记
- 搭建了 `projects/python_template/` 的初版结构
- 学会了创建虚拟环境、安装依赖、生成 `requirements.txt`
- 学会了使用分支、提交、合并和查看 Git 历史

---

## 本周遇到的问题

这一周遇到的主要问题有：

- 一开始不太理解 Git 和 GitHub 的区别
- 对 `git add`、`git commit`、`git push` 的关系容易混淆
- 刚开始不理解为什么项目要有 `.env`
- 对 `python -m src.main` 和直接运行 `python main.py` 的区别不熟
- 在 Notion 里整理 Markdown 笔记时，代码块格式偶尔会出问题

---

## 本周最重要的理解

这一周最重要的收获是：

- 开始从“写单个脚本”进入“做项目”的思维
- 理解了配置、代码、测试、文档应该分开管理
- 知道了虚拟环境和依赖管理的重要性
- 理解了 Git 不只是保存代码，而是在管理开发过程
- 意识到 README、学习笔记、周总结本身也是很重要的工程习惯
